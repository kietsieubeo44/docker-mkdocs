# Các Ví dụ về Chủ Đề

Trang này chứa nhiều ví dụ để bạn có thể thấy cách chủ đề này render các khía cạnh khác nhau của tài liệu.

## Đánh dấu mã

Dưới đây là một đoạn mã Python.

```python
a,b = 0,1
def fib():
    global a,b
    while True:
        a,b = b, a+b
        yield a

```

This is an example of inline code: `f = fib()` and `f.next()`.
